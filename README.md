# Turf Wars

Turf Wars is an open-source Roblox game inspired by the popular Turf Wars minigame on the Minecraft server, Mineplex. Players engage in team-based battles to claim territory by building structures and defeating opponents. The game features dynamic mechanics, including turf claiming, projectile combat, and block-based gameplay.

## Features

- **Dynamic Turf Management**  
  Claim turf dynamically through kills and block placement while defending your team's territory.

- **Minecraft-Inspired Mechanics**  
  Includes familiar sneaking, sprinting, and combat mechanics adapted for Roblox.

- **Team-Based Gameplay**  
  Compete in two teams with clear visual indicators for team territories and blocks.

- **Custom Tool System**  
  Players can use tools such as hammers and slingshots for building, destroying, and combat.

- **Modular and Open Source**  
  Built with maintainability in mind, leveraging best practices like modular scripts and type safety.
