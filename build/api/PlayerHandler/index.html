PlayerHandler

This item only works when running on the server. ServerAbstract

PlayerHandler is an abstract base class for managing a client's CharacterController. It provides core functionality for handling the lifecycle of the client's character, while allowing subclasses to handle specific remote events and character logic. This class cannot be instantiated directly and is intended to be extended by subclasses.
