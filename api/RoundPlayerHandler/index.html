RoundPlayerHandler

PlayerHandler

RoundPlayerHandler extends PlayerHandler to manage a client's CharacterController during a round. It is responsible for handling the client's HammerController block placement and deletion, as well as their SlingshotController projectile firing. It also manages the client's combat state and kick offenses, kicking them if deemed necessary.
