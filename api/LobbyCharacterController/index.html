LobbyCharacterController

CharacterController

LobbyCharacterController is a minimal extension of CharacterController, designed to manage player characters in the lobby. Currently, it primarily adjusts camera settings for the lobby environment but is intended to be expanded with additional lobby-specific functionality in the future.
